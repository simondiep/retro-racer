# Retro Racer

A pseudo-3d racing sandbox game.

## Development notes

This game does not use any libraries or dependencies.

Nodejs and live-server are recommended if you want to develop and see live-updates without refreshing your browser.

### To run in dev mode

```
npm i
npm start
```
